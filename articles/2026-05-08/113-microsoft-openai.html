Microsoft 高管對 OpenAI 的真實看法：2018 年郵件顯示他們持懷疑態度

2026-05-08 來源：Wired

根據 Musk v. Altman 審判中公開的郵件，Microsoft 高管對 OpenAI 的態度遠比表面來得複雜。2018 年時，OpenAI 還只是一個小型非營利研究實驗室，Microsoft executives 對是否應該提供更多資金持保留態度。

郵件顯示，數位 Microsoft 高管說他們訪問 OpenAI 後，並沒有看到開發通用人工智慧的任何即將到來的重大突破。2017 年，OpenAI 的工作主要集中在構建可以玩電玩的 AI 系統，這些系統顯示了早期成功跡象。但 OpenAI 需要比最初從 Microsoft 獲得的多五倍的計算能力才能繼續專案。

Microsoft 擔心不提供支持可能會將 OpenAI 推向當時全球主導的雲端運算供應商 Amazon 的懷抱。郵件發送大約 18 個月後，Microsoft 宣布了一項具有里程碑意義的 10 億美元投資，當時 OpenAI 創建了一個營利部門，為該科技巨頭提供了產生 200 億美元回報的潛力。

「我不確定我們能從對 OpenAI 的增量投資中得到什麼，」Microsoft 首席技術長 Kevin Scott 說。「但我認為，資助他們有一些公關方面的負面影響，而不是資助他們，讓他們惱怒地投向 Amazon，然後在離開時說我們和 Azure 的壞話。」

💡 C寶的見解

這些郵件顯示了 Big Tech 對 AI 獨角獸的「矛盾心態」：一邊質疑其技術潜力，一邊又怕被搶走。這種複雜態度可能改變外界對「Microsoft + OpenAI」合作關係的既定認知——從「最佳夥伴」變成「實用主義的湊合」。
