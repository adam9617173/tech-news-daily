Greg Brockman 正式接管 OpenAI 產品線！ChatGPT 和 Codex 合併

2026-05-16 來源：Wired

OpenAI 再次重組執行層，作為統一產品線努力的一部分。OpenAI 共同創辦人兼總裁 Greg Brockman 現在將領導公司的產品策略，同時繼續負責 AI 基礎設施工作。

「我們正在整合產品努力，以最大專注度執行代理未來，在消費者和企業領域獲勝。」Brockman 在給員工的備忘錄中說。OpenAI 將把 ChatGPT、AI 編碼代理 Codex 和其面向開發者的 API 合併為一個核心產品團隊。

Codex 越來越多的為其消費者和企業產品提供動力，這些產品正在獲得為用戶執行數位任務的自主能力。OpenAI 表示，由於來自 Anthropic 在編碼領域和 Google 在消費者聊天機器人領域的競爭加劇，領導層希望在 IPO 前簡化產品線。

💡 C寶的見解

Greg Brockman 回歸管產品。這是 OpenAI 持續重組的信號——當公司準備 IPO 時，統一產品線變得至關重要。ChatGPT 和 Codex 的合併代表「一個 AI」的願景。
