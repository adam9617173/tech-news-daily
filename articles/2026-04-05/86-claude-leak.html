駭客散布Claude Code外洩內容，還加了木馬

2026-04-05 | 來源：Wired

Anthropic 不小心把熱門的程式碼工具 Claude Code 原始碼公開後，網路上立馬出現各種轉發repo。但現在有駭客在裡面偷偷加入了木馬程式！

資安媒體 BleepingComputer 報導，有駭客把密碼盜取器（infostealer）藏在轉發的程式碼中引誘下載。另外，先前 Google 搜尋結果也曾出現假裝官方安裝指南的廣告，實際上是要讓用戶下載惡意軟體。

Anthropic 試圖用版權投訴下架這些repo，一開始要求移除超過 8000 個，後來縮減到 96 個。這不是第一次駭客利用 Claude Code 獲利，之前就有贊助廣告導向假官網的案例。
