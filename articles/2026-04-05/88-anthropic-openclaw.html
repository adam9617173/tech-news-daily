Anthropic新規：Claude Code用戶要用OpenClaw得另外付費

2026-04-05 | 來源：TechCrunch

Claude Code 的訂閱用戶未來使用 OpenClaw 等第三方工具時，可能得多花錢了！Anthropic 發出客戶信通知，從 4 月 4 日中午後，訂閱用戶「無法再使用他們的 Claude 訂閱配額用於第三方 harness（包括 OpenClaw）」，必須透過「按用量付費」的選項額外付費。

Anthropic 說這項政策「適用於所有第三方 harness」，短期內會推廣到更多工具。Claude Code 負責人 Boris Cherny 在 X 上說，公司的「訂閱系統當初不是為這些第三方工具的使用模式設計的」，現在必須「刻意管理成長，以持續永續地服務客戶」。

OpenClaw 創辦人 Peter Steinberger 之前說他要加入 OpenAI 競爭對手，結果他在 X 上說「我們試著跟 Anthropic 講道理」，但只能延後一週漲價。「真巧啊——先把他們的熱門功能 copy 到自己的 closed harness，然後就鎖住 open source」，他諷刺地說。

Cherny 強調「這比較像是工程限制問題」，並說 Anthropic 仍提供全额退款。

💡 C寶：說是「工程限制」，但時機難免讓人聯想——OpenClaw 創辦人剛跳槽 OpenAI，馬上就收費？這波操作有點微妙。不過對一般用戶影響不大，畢竟大多數人應該還是直接用官方版本。
