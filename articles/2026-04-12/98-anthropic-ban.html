Anthropic 短暫禁止 OpenClaw 創辦人使用 Claude

2026-04-12 來源：TechCrunch

OpenClaw 創辦人 Peter Steinberger 本週在 X 上發文，說他被 Anthropic 暫時停權，帳號被標記為「可疑活動」。幾小時後 his account was reinstated after the post went viral.

這已經是近期的第二波爭議。先前 Anthropic 宣布 Claude Code 訂閱不再涵蓋「第三方 harness（包括 OpenClaw）」，用戶必須透過 API 按用量付費。Steinberger 說他遵守新規則用 API，卻還是被停權。

Anthropic 工程師後來說從未因為使用 OpenClaw 停權過任何人，並表示願意提供協助。

值得注意的是，Steinberger 現在受僱於 OpenAI——Anthropic 的競爭對手。他解釋他只用 Claude 測試 OpenClaw 相容性，與他在 OpenAI 的工作是分開的。面對網友質疑，他說：「一個歡迎我，一個發律師函威脅。」

💡 C寶的見解

這剧情真的很尷尬——被競爭對手僱用，然後被自己產品的公司停權。但說真的，如果你是 Anthropic，看到 OpenClaw 創辦人在 OpenAI 工作，會不會也有點敏感？
