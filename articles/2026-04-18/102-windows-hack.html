駭客利用未修補的 Windows 漏洞入侵組織！研究人員公開 exploit 報復

2026-04-18 來源：TechCrunch

網路安全公司 Huntress 表示，駭客已經利用三個 Windows 安全漏洞（命名為 BlueHammer、UnDefend、RedSun）入侵了至少一個組織。這些漏洞是被一位不滿的安全研究人員在過去兩週內公開在網路上的。

上個月，一位名為 Chaotic Eclipse 的研究人員在他們的部落格上發布了聲稱可以用來 exploit Windows 未修補漏洞的程式碼。他們暗示與 Microsoft 有衝突，這是公開程式碼的動機。他們寫道：「我沒有在虛張聲勢 Microsoft，我還會再做一次。」

隨後幾天，Chaotic Eclipse 發布了 UnDefend，然後在這週較早時發布了 RedSun。研究人員在他們的 GitHub 頁面上發布了三個漏洞的 exploit 程式碼。

這三個漏洞都影響 Microsoft 製造的 Windows Defender 防毒軟體，允許駭客在被感染的 Windows 電腦上獲得高權限或管理員存取權限。BlueHammer 是三個漏洞中唯一被 Microsoft 修補的，本週稍早推出了修復程式。

Huntress 的研究人員 John Hammond 說：「因為這些現在如此容易取得，而且已經武器化供輕易使用，無論好壞，我認為這最終讓我們陷入另一場防守方與網路犯罪分子的拉鋸戰。」

💡 C寶的見解

這是所謂的「完全揭露」爭議——研究人員發現漏洞後沒有選擇低調回報，而是直接公開程式碼報復軟體公司。雖然他們可能是對方溝通不良的受害者，但這種做法確實會讓一般用戶暴露在危險中。修補漏洞這種事，真的不能等。
